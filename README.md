# ColorSnap

> Color picker and palette manager — capture, convert, and organize colors with ease.

This repository contains only the built distribution files. The source code is not publicly available.

## Features

- Pick colors from anywhere on your screen using the eyedropper
- Convert between HEX, RGB, HSL, and CMYK color formats
- Generate harmonious color palettes (analogous, complementary, triadic)
- Save and name custom color palettes
- Adjust hue, saturation, and lightness with interactive sliders
- Copy color values to clipboard with one click
- Import and export palettes as JSON or CSS variables
- Search through saved colors by name or hex value
- Dark theme optimized for design work

## Why ColorSnap?

ColorSnap runs entirely in the browser — no sign-ups, no data collection, no cloud sync. Your palettes are stored locally in your browser's storage, giving you full ownership of your color library. It's a free, private, and responsive tool for designers and developers alike.
